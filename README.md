# Guess exchange rate app
*Spring boot 2 academy*

## General
Simple game to guess random currency exchange ratio downloaded from [remote API](https://www.exchangerate-api.com/).

## Technologies

- Java 11
- Maven
- Spring Boot 2.4.0
- Thymeleaf - UI
